# Passport contracts

## Overview

This is the implementation of contracts for passport verification and management of passport credentials in Privado ID.

## Building contracts

1. Install dependencies:
```bash
npm i
```
2. Compile contracts:
```bash
npm run compile
```

## Testing
1. Download test circuits
```shell
./dl_circuits.sh
```

Run the different tests: 
```shell
npm test
```
